# Ejercicio America
Curso Java

## Clase 04 - Paises
Crea una colección cuyo contenido sean los países de América
Aplicando Colecciones:
	1. Listar los nombres de los países de forma ordenada alfabéticamente
	2. Implementar un método que verifique la existencia de un país
	3. Implementar un método que verifique la existencia de todos los países
	4. Implementar un método que borre un país pasado por parámetro
	5. Implementar un método que vacíe la colección
